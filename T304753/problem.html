LWF写作业

题目背景

LWF的老师布置了超多~的寒假作业，作为神，自然不能简单的糊弄了。

题目描述

LWF有 nnn 项作业 ，具体来说LWF每天会将作业按照量的多少从大到小排好序，然后选取下标为[1,n/2][1,n/2][1,n/2]的作业来做，每天做会让作业量减少 111，为了留下更多的时间van~游戏，LWF要求你求出至少 111 个作业做完后其他作业的剩余情况。

输入格式

一个整数，表示 nnn ,意义如上。

一行 nnn 个整数，表示最开始作业的量AiA_iAi​。

输出格式

一行 nnn 个整数，表示至少 111 个作业做完后其他作业的剩余情况，从大到小输出。

样例 #1

样例输入 #1

5
2 643 43 5 1

样例输出 #1

595 1 1 1 0

提示

0≤n≤5∗1030 \leq n \leq 5*10^30≤n≤5∗103

0≤Ai≤5∗1030 \leq A_i \leq 5*10^30≤Ai​≤5∗103
